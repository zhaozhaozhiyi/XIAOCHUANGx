## 剪辑节奏指引

1. **全片 BGM**：选用一首从沉稳渐强的电子氛围曲，0–20s 低频环境音为主，20s 后加入节拍，62s 后全频段推进，76s 推演段回落为思考感旋律，92s 后再次渐强，104s 全频段高潮推进，118s 回归安静
2. **转场**：所有段落间用 0.3s cross dissolve，不做花哨转场
3. **产品录屏**：实际播放速度可 1.2–1.5x 加速，但流式输出段保持原速
4. **文字动画**：大标题用弹性弹入（spring easing），正文用打字机效果，关键词亮色高亮
5. **音效**：品牌圆环出现时加柔和"叮"声；文件落入工作区时加"落定"声；PPT 翻页加轻纸声；3D 模型渲染完成加微弱"嗡"声；推演路径分叉加轻快"嗖"声
6. **调色**：全片统一暖调，产品录屏保持原有 UI 色彩，星空段偏冷对比，推演画布段略偏深蓝增加深度感
