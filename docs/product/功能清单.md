# 小窗 — 功能清单（菜单结构）

> 信息架构原则：**按用户可感知的能力划分一级模块**。  
> 能力编号（F-QA / F-WR / F-PPT / **F-RT** / **F-SET** / **F-AUTH**）保留于「能力映射」列，便于研发追溯。  
> **当前产品范围（2026-06-26）：** 导航全部可见：**对话 / 写作 / PPT / 3D绘图 / 视频 / 推演**。其中对话、写作、PPT 是 Desktop Alpha / Beta 主验收线；3D绘图、视频、推演作为 0.x 子线或 M1 能力并行推进。**翻译、会议纪要** 放到后续版本候选，不进入当前导航；**知识库** 明确不需要，不再建设独立模块。  
> **新增模块状态：** 3D绘图已有 M1 真实链路；视频已完成 [视频模块 PRD v1.0](./modules/video-module-prd.v1.md) 草案并已接入导航 / 路由 / 基座 Skill；推演已有 [推演模块 PRD v0.2](./modules/simulation-module-prd.v0.md) 与技术附录，并已接入导航 / 路由 / 基座 Skill。视频、推演当前仍属于 0.x / Beta 子线，不能按“完整业务闭环已完成”对外承诺。  
> **`0.1.0-alpha` / Desktop Alpha：** 深度验收 **对话** + **Electron 桌面壳** + Companion/CLI + 登录 + 全局设置子集。  
> **`0.2.0-beta` / Desktop Beta：** 收口 **写作 / PPT**，并补齐对话增强能力。  
> **平台边界：** 当前 Desktop = 本地文件夹工作区；Web 在线沙箱工作区为下一大版本目标，当前浏览器直连 Companion 仅视作工程态实现。
> **横切：** 账号 / 智能体 / 数据分工见主 PRD §6.7 / §6.8 / §8。切换本地项目 = **新建对话会话**。  
> **技术选型：** [docs/technical/技术方案.md](../technical/技术方案.md)。**工程：** `web/`、`companion/`、`apps/desktop/`、`api/`、`packages/*`。

## 对话 — 模式说明（页内切换，非侧栏二级）

「**快速 / 深度**」在对话页**模式选择器**中切换；侧栏二级仅保留「新对话」「历史会话」。

**混合编排（PRD F-RT-008 · 已实现）：** 平台 Push 基座 Skill + Skill Catalog 摘要 + 编排指引；写作/PPT 等扩展 Skill 与 CLI 工具由 Agent **按需**选用，无每轮强制 Router。见 [docs/technical/chat-core-architecture.md](../technical/chat-core-architecture.md)。

| 模型 API 接入 | 设置 F-SET-001/008 | `0.1.0-alpha`：OpenAI 兼容 + Anthropic；与 CLI 共用统一执行主干 |
| 模块注册表 | 平台 | 当前可见 6 个业务 `moduleId`：`chat` / `writing` / `ppt` / `3d` / `video` / `simulation`；记录领域服务、绑定键、模板资产、是否 Agent 主路径 |
| 流程 Skill | 平台 | 按 `mode` / `templateId` / `task` 绑定；非「一级菜单 = 一个 Skill」 |
| 横切规范 Skill | 平台 | `skill-platform-research-norms`，每次 Agent 任务叠加 |
| 模板资产包 | 平台 | 写作 `assets/template.*`、PPT `tpl-*`；由流程 Skill 引用，非独立 Agent 入口 |
| 工作区文件 | 横切 UI | 当前 session 的 `projectId` 根；预览、打开、定位、导出 / 另存副本；Web Sandbox 可补充下载 |
| 流式与取消 | 横切 UI | 工具链、Todo、阶段进度、中断保留 partial |
| 桌面壳 | 交付 | **`0.1.0-alpha`**：Electron 加载 `web/` + 系统选目录；**`0.2.0-beta`**：HMAC、托盘、Companion 捆绑、自动更新 |
