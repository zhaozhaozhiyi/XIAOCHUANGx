# PPT 交付自检清单

## 大纲与结构

- [ ] 页数在用户要求的区间内
- [ ] 每页有明确标题与 3～5 条要点，无空洞 filler
- [ ] 章节顺序符合路演/汇报逻辑（背景 → 现状 → 驱动 → 展望 → 风险）

## 视觉

- [ ] 仅使用选定模板的一套调色板，无混色
- [ ] 正文在 16:9 下无裁切、重叠
- [ ] 图表为静态嵌入，分辨率可读

## 数据与合规

- [ ] 数据带来源与时间
- [ ] 无捏造指标与机构观点
- [ ] 含风险提示 / 免责声明（按横切规范）

## 导出

- [ ] PPTX 在 Office / WPS 可打开
- [ ] 若经 HTML 中转，已做 fidelity 核对或说明已知差异
