## 1. 目标与范围

### 1.1 目标

将主题、写作文稿或路演模板转化为可汇报演示文稿；大纲确认 → 分页生成 → PPTX 导出；产出落 `projectId`（`XIAOCHUANG/PPT/…` 或用户课题）。

### 1.2 版本

| 阶段 | 范围 |
|------|------|
| **MVP（v3.0）** | 占位/骨架，**不验收** |
| **V1.1** | 新建 / 从文稿 / 路演模板、我的 PPT、PPTX 导出 |

## 4. 生成方式（F-PPT-001）

| 方式 | 输入 | binding |
|------|------|---------|
| 新建 PPT | 主题、受众、页数 | `{ task: "deck", templateId?: "default" }` |
| 从文稿生成 | 写作模块成稿 ID | 读工作区文稿 + deck Skill |
| 路演模板 | `templateId`（如 `pitch-deck`） | `PPT_TEMPLATE_SKILL` |

**页内流程：** 主题/选稿 → 大纲确认 → 生成幻灯片 → 预览 → 导出 PPTX。

**规则：**

- 从文稿生成：单章约 1–3 页，超长提示拆分
- 图表静态嵌入（PNG/SVG），幻灯片内不交互
