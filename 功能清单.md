# 小窗 — 功能清单（菜单结构）

> 信息架构原则：**按用户可感知的能力划分一级模块**。  
> 能力编号（F-QA / F-ARA / F-WR / F-PPT / **F-RT** / **F-SET** / **F-AUTH**）保留于「能力映射」列，便于研发追溯。  
> **MVP（v3.5，[PRD](./PRD-小窗.md) §1.2、§10.2、§12.5.9）：** 仅深度验收 **对话** + **Electron 桌面壳** + Companion/CLI + 登录 + 全局设置子集；纪要/知识库/写作/PPT/翻译 **仅占位**。收口见 [mvp-closure-checklist.md](./web/docs/mvp-closure-checklist.md)。  
> **横切：** 账号 §6.12；智能体 §6.10（F-RT-003/008/009-A 等）；数据分工 §8.6。切换本地项目 = **新建对话会话**。  
> **技术选型：** [技术方案.md](./技术方案.md)。**工程：** `web/`、`companion/`、`apps/desktop/`、`api/`、`packages/*`。

## 对话 — 模式说明（页内切换，非侧栏二级）

「**快速 / 深度**」在对话页**模式选择器**中切换（页面上不区分深度思考与深度研究）；侧栏二级仅保留「新对话」「历史会话」。

**混合编排（PRD v3.5 F-RT-008 · 已实现）：** 平台 Push 基座 Skill + Skill Catalog 摘要 + 编排指引；PPT/写作等扩展 Skill 与 CLI 工具由 Agent **按需**选用，无每轮强制 Router。见 [web/docs/chat-core-architecture.md](./web/docs/chat-core-architecture.md)、PRD §12.5.9。

## 写作 — 生成流程（非菜单，页内能力）

进入任一类模板或「新建写作」后，用户可选择：

| 流程 | 步骤 |
|------|------|
| 多步骤 | 参数设置 → 明确写作方向 → 大纲确认 → 撰写 |
| 快速 | 最简参数 → 大纲确认（可跳过）→ 撰写 |

## 智能体与工作区（横切）

| 功能项 | 类型 | 说明 |
|--------|------|------|
| 模式 B 本地协作 | 主路径 | Companion + projectId（沙箱或 local_bound）+ 本机 CLI |
| 沙箱项目 | 模式 B 默认 | 新建对话未选文件夹 → Companion 托管目录 |
| 云端项目 | 模式 A 默认 | 无 Companion → 新建对话自动 cloud + OSS |
| 本地绑定项目 | 能力 | **文件夹导入** → `baseDir` 原目录读写；须**新建对话会话**绑定（F-RT-007c） |
| 添加新项目（Web） | 交互 | 路径输入 `~/Projects/...`；禁止 Directory Picker 主路径 |
| 添加新项目（桌面） | 交互 | Electron 系统选目录；Web 仅见 `projectId`（V1.1） |
| 会话固定 projectId | 规则 | 创建时绑定不可改；换文件夹=新建会话；分支继承 projectId |
| 模式 A 云端子集 | MVP | 见 PRD §10.2（对话 + 文件树；写作/PPT 可收缩） |
| 平台 CLI 适配集 | 平台 | 多 CLI / 多模型适配；IT 桌面镜像或 Companion 捆绑 |
| 企业扩展 CLI | 策略 | 企业扩展表登记；Web 无任意 exe 路径 |
| 默认 Agent 选择 | 设置 F-SET-001 | 列表内切换；对话区「进入项目工作」 |
| Companion / CLI 状态 | 设置 F-SET-001 | 连接、可用 / 未安装 / 需登录 / 版本过低 |
| 模型 API 接入 | 设置 F-SET-001/008 | MVP：OpenAI 兼容 + Anthropic；与 CLI 共用统一执行主干 |
| 模块注册表 | 平台 | 六个 `moduleId` → 领域服务、绑定键、流程 Skill、模板资产、是否 Agent 主路径（PRD §6.10.1a） |
| 流程 Skill | 平台 | 按 `mode` / `templateId` / `task` 绑定；非「一级菜单 = 一个 Skill」 |
| 横切规范 Skill | 平台 | `skill-platform-research-norms`，每次 Agent 任务叠加 |
| 模板资产包 | 平台 | 写作 `assets/template.*`、PPT/纪要 `tpl-*`；由流程 Skill 引用，非独立 Agent 入口 |
| 工作区文件 | 横切 UI | 当前 session 的 projectId 根；预览、下载、导出 |
| 流式与取消 | 横切 UI | 工具链、Todo、阶段进度、中断保留 partial |
| 桌面壳 | 交付 | **MVP**：Electron 加载 `web/` + 系统选目录；**V1.1**：HMAC、托盘、Companion 捆绑、自动更新 |
